# 2048 Game

A classic sliding tile puzzle game where you combine matching numbers to reach 2048.

## How to Play

- Use **arrow keys** (desktop) or **swipe** (mobile) to move all tiles in one direction
- When two tiles with the same number collide, they merge into one tile with their sum
- After each move, a new tile (2 or 4) appears in a random empty space
- The game ends when no more moves are possible
- Try to reach the 2048 tile!

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Touch Support**: Swipe gestures for mobile play
- **Score Tracking**: Current score and best score persistence
- **Smooth Animations**: Tile appearance and merge animations
- **Game Over Detection**: Automatic game over detection with restart option
- **Multi-language**: English and Chinese language support

## Tech Stack

- Pure HTML5
- CSS3 (Grid layout, animations, responsive design)
- Vanilla JavaScript (no frameworks)
- LocalStorage for best score persistence
